## 📌 Project Overview

This project was completed as part of the **SyntecxHub Data Analysis Internship**.

The objective was to analyze retail sales data, identify key business trends, and develop an interactive Power BI dashboard to support data-driven decision-making.

---

## 🎯 Objectives

- Clean and preprocess sales data
- Analyze monthly and yearly sales trends
- Measure business KPIs
- Identify top-performing products
- Compare regional performance
- Build an interactive Power BI dashboard
- Generate actionable business insights

---

## 🛠 Tools Used

- Microsoft Power BI
- Microsoft Excel
- SQL
- Python
- GitHub

---

## 📂 Dataset

The dataset includes:

- Order ID
- Order Date
- Product
- Category
- Region
- Segment
- Sales
- Profit
- Quantity

---

## 📊 Dashboard Features

- Total Sales KPI
- Total Profit KPI
- Total Orders
- Profit Margin
- Monthly Sales Trend
- Category Analysis
- Region Analysis
- Top 10 Products
- Interactive Filters

---

## 📈 Business Insights

- Technology generated the highest revenue.
- West region achieved the highest sales.
- Quarter 4 recorded the strongest business performance.
- High-margin products significantly contributed to profitability.

---

## 💡 Recommendations

- Increase stock for top-selling products.
- Focus marketing efforts on underperforming regions.
- Optimize pricing strategies.
- Monitor KPIs regularly for business growth.
